# 🌌 Agentic OS: System Capabilities

The Agentic OS is a decentralized, multi-agent orchestration framework designed for complex problem-solving through specialized workers and RAG-integrated skills.

## 🛠 Core Specialist Roles
I can spin up and coordinate the following specialized agents to handle your requests:

| Role | Responsibility |
| :--- | :--- |
| **RAG** | Vector-search based retrieval across the internal knowledge base and skill library. |
| **RESEARCH** | Autonomous web searching, document scraping, and competitive intelligence gathering. |
| **CODE** | Arbitrary code execution in a secure, sandboxed container (Tools API). |
| **SCHEMA** | SQL-aware database introspection and capability discovery. |
| **PLANNER** | High-level task decomposition and iterative refinement of execution steps. |
| **EXECUTOR** | General-purpose ReAct reasoning for multi-hop tool usage. |
| **EMAIL** | SMTP/IMAP integration for communication workflows. |
| **PRODUCTIVITY** | Note-taking, TODO management, and calendar integration. |

## 🧠 Specialized Skill Library
I have access to over **260+ pre-vetted AI skills** stored in `assets/skills/skills`. These skills provide the "know-how" for specific domains:

- **Engineering**: Architecture design, CI/CD pipeline building, security auditing (ISO27001), and TDD workflows.
- **Business Operations**: C-Level advisors (CEO, CTO, CFO), finance analysis, and churn prevention.
- **Marketing & Growth**: SEO auditing, ad creative generation, and competitive teardowns.
- **Product Management**: PRD authoring, Agile coaching, and user story generation.

## 🧩 Integrated Toolsets
My agents can interact with the physical world through the following tool registries:

- **System Tools**: File system access, shell commands (sandboxed), and process management.
- **Database Tools**: Postgres introspection and PgVector similarity searching.
- **MCP (Model Context Protocol)**: Support for Brave Search, GitHub API, Google Workspace, and Puppeteer.
